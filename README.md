# 📋 LaTrabajadera

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=for-the-badge&logo=firebase&logoColor=white)

**LaTrabajadera** es una aplicación diseñada para optimizar, organizar y mapear de forma dinámica la disposición y el orden de los costaleros debajo de los pasos de Semana Santa. La herramienta permite gestionar las posiciones de forma personalizada basándose en cuadrantes y alturas.

## ✨ Características Principales
* **Mapeo Dinámico:** Diseño personalizado de la estructura del paso y asignación visual de puestos para los costaleros.
* **Sincronización en Tiempo Real:** Gestión de datos centralizada mediante Firebase para asegurar que los cambios en el cuadrante se actualicen al instante.
* **Gestión de Alturas y Relevos:** Facilita la organización del personal en las trabajaderas optimizando la distribución según las características de cada costalero.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje de Programación:** Java
* **Base de Datos y Backend:** Firebase (Realtime Database / Cloud Firestore)
* **Entorno de Desarrollo:** Android Studio / Java SE *(Modifica según corresponda)*
